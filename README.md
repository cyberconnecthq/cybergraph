# cybergraph

# Set Up

0. Upgrade your foundry
   `foundryup`

1. To enable husky pre-commit
   `yarn && yarn prepare`

2. To install contract dependencies
   `forge install`

3. To build
   `forge build`

4. To test
   `forge test -vvv`

5. (optional) To run static analysis
   `slither src/`

6. To see contract sizes
   `yarn size`
